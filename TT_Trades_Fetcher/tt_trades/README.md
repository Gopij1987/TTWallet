# TT Trades - Trade Data Extraction for Tradetron

Reusable shared code for extracting trade data from Tradetron using their API with stored cookies.

## Files

### Core Modules

- **`tradetron_api.py`** - Main API client for authenticated requests to Tradetron
  - `TradetronAPIClient` class with methods for all common endpoints
  - Handles authentication using stored cookies
  - Includes connection testing and error handling

- **`trade_extractor.py`** - Trade data extraction and formatting
  - `TradeDataExtractor` class for extracting and formatting trade data
  - CSV and JSON export functions
  - Trade summary and statistics calculation
  - Batch extraction with pagination

- **`examples.py`** - Usage examples and quick-start guides
  - 6 different examples showing how to use the modules
  - Copy and modify for your specific needs

## Quick Start

### 1. Test Connection

```python
from tradetron_api import TradetronAPIClient

client = TradetronAPIClient()
client.test_connection()
```

### 2. Extract All Trades

```python
from trade_extractor import TradeDataExtractor

extractor = TradeDataExtractor()
trades = extractor.extract_all_trades(limit=1000)
summary = extractor.get_trade_summary(trades)
extractor.print_summary(summary)
```

### 3. Export to File

```python
# Export to JSON
extractor.export_to_json("trades_data.json")

# Export to CSV
extractor.export_trades_to_csv("trades_data.csv")
```

### 4. Get Trades for Specific Strategy

```python
client = TradetronAPIClient()
trades = client.get_strategy_trades(strategy_id=18713274, limit=100)
```

## API Client Methods

### Available Endpoints

- `test_connection()` - Test if connection to Tradetron works
- `get_strategies()` - Get all strategies
- `get_strategy_details(strategy_id)` - Get specific strategy info
- `get_all_trades(limit, offset)` - Get all trades with pagination
- `get_strategy_trades(strategy_id, limit, offset)` - Get trades for a strategy
- `get_portfolio()` - Get portfolio information
- `get_orders(limit)` - Get recent orders
- `get_dashboard()` - Get dashboard data

## Requirements

- Python 3.7+
- `requests` library (installed in venv)
- Valid `.env` file with `TT_COOKIES_B64_GOPI` (generated by `refresh_cookies_TTGopiWallet.py`)

## Data Files Generated

When you run extraction, files are saved in the `tt_trades/` folder:

- `trades_YYYYMMDD_HHMMSS.json` - Raw trade data as JSON
- `trades_YYYYMMDD_HHMMSS.csv` - Trade data as CSV
- `complete_report_YYYYMMDD_HHMMSS.json` - Full report with metadata

## Reusing in Other Projects

To use these modules in TTextractor or other projects:

1. Copy the core modules (or reference this folder):
   ```python
   import sys
   from pathlib import Path
   sys.path.insert(0, str(Path(__file__).parent / "../../TT_Trades_Fetcher/tt_trades"))
   
   from tradetron_api import TradetronAPIClient
   from trade_extractor import TradeDataExtractor
   ```

2. Or import directly if in same virtual environment:
   ```python
   from tt_trades.tradetron_api import TradetronAPIClient
   from tt_trades.trade_extractor import TradeDataExtractor
   ```

## Authentication

The modules automatically:
1. Load cookies from `.env` file
2. Validate cookies before making requests
3. Handle expired cookies (tells you to refresh)

To refresh cookies when they expire:
```bash
cd ..
python refresh_cookies_TTGopiWallet.py
```

## Error Handling

- Expired cookies: You'll see "Unauthorized (401)" - run refresh_cookies script
- No trades found: Check if strategy has trades or adjust limit
- Connection errors: Check internet connection and verify .env cookies are valid

## Tips

- Start with `limit=100` to test, then increase for full data extraction
- Use pagination (offset) for large datasets to avoid memory issues
- Batch export to JSON/CSV alongside extraction for large datasets
- Keep `.env` file secure - it contains authentication tokens

## Common Use Cases

### Extract weekly performance report
```python
extractor = TradeDataExtractor()
trades = extractor.extract_all_trades()
summary = extractor.get_trade_summary(trades)
extractor.export_trades_to_csv(f"weekly_report_{datetime.now().date()}.csv")
```

### Monitor specific strategy
```python
client = TradetronAPIClient()
strategy_trades = client.get_strategy_trades(strategy_id=YOUR_ID)
# Process trades...
```

### Automated daily extraction
```python
def daily_extraction():
    extractor = TradeDataExtractor()
    trades = extractor.extract_all_trades()
    date = datetime.now().strftime("%Y%m%d")
    extractor.export_trades_to_csv(f"trades_{date}.csv")
```
